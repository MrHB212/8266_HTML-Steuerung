# 8266-Konfiguration über HTML (POST-GET)


Ein Projekt mit dem Ziel einem ESP8266 über eine HTML-Datei Anweisungen und Daten zu schicken. Es wird angenommen, dass das ESP sich im gleichen Netzwerk befindet.
